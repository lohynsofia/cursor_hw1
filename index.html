Basic commands:

  1. git config --global user.name "name" - input user name;
  2. git config --global user.email "email" - input user email;
  3. git init - creating the repository;
  4. git status - output information about the repository (status -s - short information); for more info we can use "git diff";
  5. git add - adding the file under control before commiting (add . - add all files in folder; git remote add origin LINK - add remote repository to GIT);
  6. git commit - save changes from local files to repository (commit -m "comment" - adding comment about commit; commit --amend - cancelling the commit);
  7. git log - output the history of all commits;
  8. git checkout - check/move to certain commit (checkout - move to; checkout - - move to previous commit);
  9. git branch - create branch from main commit;
  10. git push - saving changes from local to remote repository (origin "branch/main name" - saving from certain branch, after this command we can use only push);
  11. git pull - download changes from remote to local repository;
  12. git merge - connect two versions from different branches;
  13. git clone - copy existing repository;
  14. .gitignore - ignore some files while adding all files (for example: *.[a] - ignore all files, names of which end on "a");
  15. ls - show files in the folder;
  16. cd - move Git Bash to folder.

Anvanced:

  1. git revert - turns back to commit and allows you to change it;
  2. git reset HEAD - cancelling command "git add";
  3. git rebase - moves your branch from one to another commit;
  4. git cherry-pick - takes the commit from one branch and move to another;
  5. git stash - creates tentative version of your work without connecting to main working branch.
